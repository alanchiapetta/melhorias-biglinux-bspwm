Atalhos do BSPWM

Terminal e Launcher

Terminal (Alacritty)

  • Super + Enter: Terminal em modo janela lado a lado
  • Super + Shift + Enter: Terminal em modo Flutuante
  • Super + Alt + Enter: Terminal em modo Tela Cheia

Rofi (Launcher)

  • Super ou Alt + F1: Lançador de aplicativos
  • Super + x: Power Menu
  • Super + s: Menu de PrintScreen
  • Super + w: Lista de janelas abertas
  • Super + d: Pesquisar e executar (dmenu)

Programas

Programas principais

  • Super + Shift + k: Lista de Atalhos BSPWM
  • Super + Shift + f: Gerenciador de Arquivos (PcmanFM)
  • Super + Shift + w: Browser (Firefox)
  • Super + Shift + e: Editor de Textos (Geany)

Programas via terminal

  • Super + Shift + m: Player de Música (cmus)
  • Ctrl + Alt + v: Editor de Textos (nvim)
  • Ctrl + Alt + r: Gerenciador de Arquivos (ranger)
  • Ctrl + Alt + h: Monitor de Sistema (htop)

Sistema

Atalhos do Sistema

  • Super + C: Fechar janela
  • Ctrl + Alt + Esc: Matar janela (xkill)
  • Ctrl + Shift + q: Logoff
  • Ctrl + Shift + r: Reiniciar BSPWM
  • Super + Esc: Recarregar atalhos

Workspaces

  • Ctrl + Alt + Esquerda: Ir ao Workspace à esquerda
  • Ctrl + Alt + Direita: Ir ao Workspace à direita
  • Super + 1-0: Ir ao Workspace 1-10
  • Super + Shift + 1-0: Move janela para Workspace 1-10
  • Super + Ctrl + Shift + Esquerda: Move janela para Workspace à esquerda
  • Super + Ctrl + Shift + Direita: Move janela para Workspace à direita

Janelas

  • Super + Esquerda/Direita/Baixo/Cima: Muda o foco para a janela à esquerda/direita/abaixo/acima
  • Super + Shift + Esquerda/Direita/Baixo/Cima: Troca janela de posição para esquerda/direita/abaixo/acima
  • Alt + Shift + Esquerda/Direita/Baixo/Cima: Movimenta janelas flutuantes
  • Super + Ctrl + Esquerda/Direita/Baixo/Cima: Expande tamanho da janela
  • Super + Alt + Esquerda/Direita/Baixo/Cima: Encolhe tamanho da janela
  • Super + h: Divide janela na horizontal
  • Super + v:Divide janela na vertical
  • Super + q: Cancela divisão da janela
  • Super + f: Alterna janela para modo Tela Cheia
  • Super + Espaço: Alterna janela entre os modos Flutuantes e Lado a Lado
  • Alt + Tab: Alterna o foco para a próxima janela
  • Alt + Shift + Tab: Alterna o foco para a janela anterior
